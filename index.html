Introduction

El Maven is an open source, vendor neutral, LC-MS data processing engine that is most optimal for isotopomer labeling and untargeted metabolomic profiling experiments. Currently Maven exists as a desktop application that runs on Windows and Ubuntu. The aim of this software package is to reduce complexity of metabolomics analysis by using a highly intitute interface for exploring and validating metabolomics data.

History

The birth of El Maven can be traced back to another open source project, Maven, an open source LC-MS data processing engine developed by Euguene Melamud. We decided to use El Maven as the working title for the current project for few reasons. One, we wanted to acknowledge the original project. We also want to commit ourselves to create “The” version of Maven that is widely used and supported by the community. Finally, El also bears the stamp of Elucidata.
