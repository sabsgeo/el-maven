History of Maven

Installation on windows

Installing MSYS2

Download the MSYS2 installer.

Click and download 32 bit or 64 bit windows installer.

Run the installer by double clicking the installer.

Select the folder in which you want to install MSYS2 and then ‘Next’. After the installation completes, tick ‘Run MSYS2 now’ and click ‘Finish’.

Update the package database and pacman with: 

pacman -Sy

If needed, close MSYS2, run it again from the installation folder by double clicking 'msys2.exe'.
Update the package database and core system packages with: 

pacman -Syu

Again, if needed, close MSYS2, run it again from the installation folder. Update the rest with: 

pacman -Su

Install the MSYS2 development packages:

- base-devel contains basic development tools for any building. Install it using the command: 

pacman -Sy base-devel

- msys2-devel package contains tools for building msys2 packages. Install it using the command: 

pacman -Sy msys2-devel

- mingw-w64-x86_64 is a toolchain used to build mingw64 packages. Install it using the command: 

pacman -Sy mingw-w64-x86_64-toolchain

- mingw-w64-x86_64-qt4 - The Qt Installer Framework used for the Qt SDK installer. Install QT4 using the command: 

pacman -Sy mingw-w64-x86_64-qt4

hdf5 is a general purpose library and file format for storing scientific data. Install it using the command: 

pacman -Sy mingw64/mingw-w64-x86_64-hdf5

Interface for scientific data access to large binary data. Install it using the command: 

pacman -Sy mingw64/mingw-w64-x86_64-netcdf

This completes the installation of MSYS2 and the tool chain required to build Maven on your computer.

The next step is to get the Maven from the repository.

Install GIT on your system by typing the command in the MSYS2 terminal: 

pacman -Sy msys/git

Create a folder on Windows. Navigate to that folder using cd command: 

cd <pathtofolder>

For Example: 

cd /c/User/Admin/Desktop

To initialise a git repository use: 

git init
